Here you can find any data used for the GEO2253 Story Map. 
Sources have been given in the Story Map itself. 

Metadata of the Landsat Data is the following:
Cam LS7 image has been taken on 13/09/2002 by Landsat 7, Bands 1, 2 and 3 have been used to gain a RGB image.
Cam LS8 image has been taken on 06/10/2022 by Landsat 8, Bands 2, 3 and 4 have been used to gain a RGB image.
Derwent LS7 image has been taken on 20/09/2002 by Landsat 7, Bands 1, 2 and 3 have been used to gain a RGB image.
Derwent LS8 image has been taken on 17/09/2022 by Landsat 8, Bands 2, 3 and 4 have been used to gain a RGB image.
